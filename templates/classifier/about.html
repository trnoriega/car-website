About CarMatch

CarMatch was conceived and developed during three weeks while I was a fellow with Insight Data Science.
It uses an Inception Neural Network implented in Keras (with a TensorFlow backend) to identify 196 different car classes.

The Github repo for the network training experiments can be found here.
The Github repo for the Django frontend can be found here.
